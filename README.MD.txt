# Data Visualization Project - Sprint 4

## Project Summary:
This project includes various analyses on product profitability, return rates, and advertising recommendations for a superstore.

### Included Visualizations:
1. **Profit Analysis:**
   - Biggest Profit Centers and Biggest Loss Producers.
   
2. **Return Rate Analysis:**
   - Product and Customer Return Rates.
   
3. **Advertising Recommendations:**
   - State and Monthly Analysis for Best Advertising Opportunities.
